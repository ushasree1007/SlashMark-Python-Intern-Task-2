I’m excited to share that I have successfully completed Task 2 – Number Guessing Game using Python.

📌 Project Description
This is a simple and interactive number guessing game where the computer randomly selects a number between 1 and 200, and the player has to guess the correct number using hints provided by the system.

The game gives responses such as:
Too High
Too Low
Correct Guess
It also validates user input and counts the total number of attempts.

🛠 Technologies Used
Python
Random Module
Conditional Statements
Loops

✨ Features
Random number generation
User-friendly console interaction
